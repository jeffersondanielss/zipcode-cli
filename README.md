# Zipcode CLI

## Usage

```bash
  $ zipcode <cep>
```

### Options

Show just address, `-a, --address`.

```bash
$ zipcode <cep> -a
```

Show just city, `-c, --city`.

```bash
$ zipcode <cep> -c
```
  
Show just state, `-s, --state`.

```bash
$ zipcode <cep> -s
```

Show just district, `-s, --district`.

```bash
$ zipcode <cep> -d
```
